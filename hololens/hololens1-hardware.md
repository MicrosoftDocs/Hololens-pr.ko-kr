# <a name="hololens-1st-gen-hardware"></a>HoloLens (첫 번째 gen) 하드웨어

![Microsoft HoloLens (첫 번째 gen)](images/see-through-400px.jpg)

Microsoft HoloLens (첫 번째 gen)는 세계 최초의 완전히 작업할 holographic 컴퓨터입니다. HoloLens는 holographic 환경을 통해 개인 컴퓨팅을 다시 정의 하 여 새로운 방식으로 역량을 강화 합니다. HoloLens는 최첨단 광학 및 센서를 혼합 하 여 실제 세계에 고정 된 3D holograms을 제공 합니다.

## <a name="hololens-components"></a>HoloLens 구성 요소

![상자의 내용을 표시 하는 이미지](images/hololens-box-contents.png)

HoloLens에는 다음과 같은 기능이 포함 되어 있습니다.

- **센터**. HoloLens 센서 및 표시를 포함 합니다. HoloLens를 입고 있는 동안에는 센터를 회전할 수 있습니다.
- **헤드 밴드**. HoloLens를 설정 하려면 조정 휠을 사용 하 여 헤드 밴드를 확장 합니다. HoloLens가 제자리에 있으면 헤드 밴드가 편안 해질 때까지 조정 휠을 강화 합니다.
- **밝기 단추**. HoloLens를 사용할 때 밝기 단추는 장치의 왼쪽에 있습니다.
- **볼륨 단추**. HoloLens를 입고 있으면 볼륨 단추가 장치 오른쪽에 표시 됩니다.
- **장치 arm**. HoloLens를 선택 하거나, 설정 하거나, 사용 하는 경우, 언제 든 지 장치 arm을 사용 하 여 사용 하거나 저장할 수 있습니다.

## <a name="in-the-box"></a>상자에서

> [!VIDEO https://www.microsoft.com/videoplayer/embed/c7ceb904-9d5f-4194-9e10-e8a949dbad7d]  

HoloLens 상자에는 다음 항목이 포함 됩니다.

- **코 패드**. 코의 모양에 맞는 코 패드를 선택 하 고 eyewear를 수용 합니다.
- **오버 헤드 스트랩을**. 이동 하는 동안 HoloLens를 사용 하는 경우 오버 헤드 스트랩을을 사용 하 여 장치를 유지 하는 데 도움이 됩니다. 또한 확장 된 기간 동안 HoloLens를 스트랩을 오버 헤드를 사용 하 여 장치를 더 편안 하 게 만들 수 있습니다.
- **마이크로 USB 케이블** 마이크로 USB 케이블을 사용 하 여 장치를 충전을 위해 전원 공급 장치에 연결 하거나, 컴퓨터에 HoloLens를 연결 하는 데 사용 합니다.
- **전원 공급 장치**. 전원 콘센트에 연결 합니다.
- **마이크로 파이버 천으로** 천을 사용 하 여 HoloLens 센터를 정리 합니다.

### <a name="power-supply-details"></a>전원 공급 장치 세부 정보

장치와 함께 제공 되는 전원 공급 장치 및 USB 케이블은 요금을 청구 하는 데 가장 적합 한 메커니즘입니다. 전원 공급 장치는 18W 충전기입니다.  2A에서 9V를 제공 합니다.

요금 청구 속도와 속도는 장치가 실행 되는 환경에 따라 달라질 수 있습니다.

장치가 켜져 있는 동안 내부 배터리 충전 비율을 유지 하기 위해 최소 15W 충전기에 연결 해야 합니다.

## <a name="care-and-cleaning"></a>주의 하 고 정리

HoloLens를 신중 하 게 처리 합니다. HoloLens를 리프트 하 고, 운반 하 고, 조정 하려면 &mdash; 오버 헤드 스트랩을 없는 장치 arm을 사용 합니다. 센터를 먼지와 지문을 사용 하지 않고 유지 하 고 접촉 하지 않도록 합니다. 반복 되는 청소가 센터를 손상 시킬 수 있으므로 장치를 깨끗 하 게 유지 해 보세요.

HoloLens에서 클리너 또는 고 지를 사용 하지 않고 물에 직접 병합 하거나 물에 직접 적용 하지 않습니다.

센터를 청소 하려면 머리카락 렌즈 브러시 또는 전구 스타일 렌즈 blower에서 카멜식 나 goat 사용 하 여 먼지를 제거 합니다. 적은 양의 distilled 물로 마이크로 파이버 천을 moisten 하 고이를 사용 하 여 원형 동작에서 센터를 천천히 지울 수 있습니다.

헤드 밴드 및 장치 팔을 포함 하 여 나머지 장치를 청소 하려면 가벼운 soap 및 물이 있는 보풀이 없는 천으로 moistened를 사용 합니다. 다시 사용 하기 전에 HoloLens를 완전히 건조 해 보세요.

## <a name="finding-the-serial-number"></a>일련 번호 찾기

HoloLens (첫 번째 gen) 장치의 일련 번호는 오른쪽 렌즈 아래에 인쇄 됩니다.
1. 사용자가 원하는 대로 장치를 최대한 활용 하세요. 장치를 앞으로 이동 하 고 수직으로 연결 합니다.
1. 계속 해 서 렌즈를 사용 하 여 장치를 뒤집어 전환 하면 아래쪽이 표시 됩니다.
1. 이제 오른쪽 렌즈가 왼쪽에 표시 되 고, 대역의 렌즈 아래에는 일련 번호가 표시 됩니다. 

연결 된 PC를 통해 일련 번호를 찾을 수도 있습니다.

1. 장치 플러그 인
1. 파일 탐색기에서 **이 PC** 로 이동 합니다.
1. 마우스 오른쪽 단추를 클릭 하 고 HoloLens 장치의 **속성** 을 선택 합니다.
1. 그러면 장치의 일련 번호가 표시 됩니다. 아래 스크린샷에서와 같습니다.
