# <a name="hololens-1st-gen-hardware"></a>HoloLens(1세대) 하드웨어

![Microsoft HoloLens (첫 번째 gen)](images/see-through-400px.jpg)

Microsoft HoloLens (첫 번째 gen)은 세계 최초의 완전히 작업할 holographic 컴퓨터입니다. holographic 환경을 통해 개인 컴퓨팅을 다시 정의 하 여 새로운 방식으로 역량을 강화 HoloLens. HoloLens는 최첨단 광학 및 센서를 혼합 하 여 실제 세계에 고정 된 3d holograms를 제공 합니다.

## <a name="hololens-components"></a>HoloLens 구성 요소

![상자의 내용을 표시 하는 이미지](images/hololens-box-contents.png)

HoloLens에는 다음과 같은 기능이 포함 되어 있습니다.

- **센터**. HoloLens 센서와 표시를 포함 합니다. HoloLens를 입고 있는 동안에는 센터를 회전할 수 있습니다.
- **헤드 밴드**. HoloLens를 배치 하려면 조정 휠을 사용 하 여 헤드 밴드를 확장 합니다. HoloLens 현재 위치가 설정 된 상태에서 헤드 밴드가 편안 해질 때까지 조정 휠을 강화 합니다.
- **밝기 단추**. HoloLens를 입고 있는 경우 밝기 단추는 장치의 왼쪽에 있습니다.
- **볼륨 단추**. HoloLens를 입고 있으면 볼륨 단추가 장치의 오른쪽에 있습니다.
- **장치 arm**. HoloLens를 선택 하거나, 사용 하는 경우, 장치 arm에서 항상이를 잡고 보유 합니다.

## <a name="in-the-box"></a>상자에서

> [!VIDEO https://www.microsoft.com/videoplayer/embed/c7ceb904-9d5f-4194-9e10-e8a949dbad7d]  

HoloLens 상자에는 다음 항목이 포함 되어 있습니다.

- **코 패드**. 코의 모양에 맞는 코 패드를 선택 하 고 eyewear를 수용 합니다.
- **오버 헤드 스트랩을**. 이동 하는 동안 HoloLens를 사용 하는 경우 오버 헤드 스트랩을를 사용 하 여 장치를 유지 하는 데 도움이 됩니다. 또한 스트랩을를 사용 하 여 확장 된 기간 동안 HoloLens를 작성 하는 경우 오버 헤드를 사용 하면 장치를 더욱 편안 하 게 만들 수 있습니다.
- **마이크로 USB 케이블** 마이크로 USB 케이블을 사용 하 여 HoloLens 전원 공급 장치에 연결 하거나,이를 사용 하 여 HoloLens을 컴퓨터에 연결 합니다.
- **전원 공급 장치**. 전원 콘센트에 연결 합니다.
- **마이크로 파이버 천으로** 천을 사용 하 여 HoloLens 센터를 정리 합니다.

### <a name="power-supply-details"></a>전원 공급 장치 세부 정보

장치와 함께 제공 되는 전원 공급 장치 및 USB 케이블은 요금을 청구 하는 데 가장 적합 한 메커니즘입니다. 전원 공급 장치는 18W 충전기입니다.  2A에서 9V를 제공 합니다.

요금 청구 속도와 속도는 장치가 실행 되는 환경에 따라 달라질 수 있습니다.

장치가 켜져 있는 동안 내부 배터리 충전 비율을 유지 하기 위해 최소 15W 충전기에 연결 해야 합니다.

## <a name="care-and-cleaning"></a>주의 및 정리

HoloLens 신중하게 처리합니다. HoloLens 리프트, 캐리 및 조정하려면 &mdash; 오버헤드가 아닌 디바이스 암(arms)을 사용합니다. visor를 더 이상 사용하지 않도록 하고 지문을 사용하지 않도록 합니다. 청소를 반복하면 visor가 손상되어 디바이스를 정리해 보세요.

HoloLens 클리너 또는 용매를 사용하지 말고, 물로 클리너나 용매를 직접 적용하지 마세요.

바이저를 정리하려면 카멜식 또는 빗살 무백 브러시 또는 전구 스타일 렌즈 렌더를 사용하여 모든 미세먼지를 제거합니다. 소량의 재배된 물로 마이크로 피버 의류를 약간 습성화한 다음, 이를 사용하여 원을 그리며 바이저를 초기화합니다.

헤드 밴드 및 디바이스 암을 포함하여 디바이스의 나머지 부분을 정리하려면 깔끔하고 물과 함께 보습된 lint-free 의류를 사용합니다. 다시 사용하기 전에 HoloLens 완전히 말립니다.

## <a name="finding-the-serial-number"></a>일련 번호 찾기

HoloLens(1세대) 디바이스의 일련 번호는 오른쪽 렌즈 아래에 인쇄됩니다.
1. 머리 위에 디바이스를 쓰려는 것처럼 디바이스를 앞에 붙들고 있습니다. 디바이스가 앞으로 및 수직으로 향합니다.
1. 렌즈가 계속 앞으로 향하면 아래쪽을 볼 수 있도록 디바이스를 아래로 대칭 전환합니다.
1. 오른쪽 렌즈는 이제 왼쪽에 있으며, 밴드에서 렌즈 아래에 인쇄되는 일련 번호입니다. 

연결된 PC를 통해 일련 번호를 찾을 수도 있습니다.

1. 디바이스 플러그 인
1. 파일 탐색기에서 **이 PC로 이동합니다.**
1. 마우스 오른쪽 단추를 클릭하고 HoloLens 디바이스의 **속성을** 선택합니다.
1. 그러면 디바이스의 계열 번호가 표시됩니다. 아래 스크린샷과 같습니다.
