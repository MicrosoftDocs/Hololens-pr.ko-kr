# <a name="hololens-1st-gen-hardware"></a>HoloLens(1세대) 하드웨어

![Microsoft HoloLens(1세대).](images/see-through-400px.jpg)

Microsoft HoloLens(1세대)는 완전히 테더링되지 않은 세계 최초의 홀로그램 컴퓨터입니다. HoloLens 홀로그램 환경을 통해 개인 컴퓨팅을 다시 정의하여 새로운 방식으로 역량을 강화합니다. HoloLens 최첨단 광학 및 센서를 혼합하여 주변 현실 세계에 고정된 3D 홀로그램을 제공합니다.

## <a name="hololens-components"></a>HoloLens 구성 요소

![상자의 내용을 보여 주는 이미지입니다.](images/hololens-box-contents.png)

HoloLens 다음과 같은 기능이 포함되어 있습니다.

- **바이저**. HoloLens 센서 및 디스플레이를 포함합니다. HoloLens 동안 바이저를 위로 회전할 수 있습니다.
- **헤드밴드**. HoloLens 켜려면 조정 휠을 사용하여 헤드 밴드를 확장합니다. HoloLens 있는 경우 헤드 밴드가 편안하게 될 때까지 조정 휠을 강화합니다.
- **밝기 단추**. HoloLens 사용하면 밝기 단추가 디바이스의 왼쪽에 있습니다.
- **볼륨 단추**. HoloLens 눌리면 볼륨 단추가 디바이스의 오른쪽에 있습니다.
- **디바이스 암(arms)** HoloLens 픽업, 켜기 또는 끄면 항상 디바이스를 품에 안거나 붙들고 있습니다.

## <a name="in-the-box"></a>상자 안

> [!VIDEO https://www.microsoft.com/videoplayer/embed/c7ceb904-9d5f-4194-9e10-e8a949dbad7d]  

HoloLens 상자에는 다음 항목이 포함됩니다.

- **코패드.** 코 모양에 맞고 눈 모양에 맞는 코 패드를 선택합니다.
- **오버헤드 스트랩** . 이동하는 동안 HoloLens 사용하는 경우 오버헤드를 사용하여 디바이스를 제자리에 유지합니다. 또한 장시간 HoloLens 사용하는 경우 오버헤드 로체를 사용하면 디바이스를 더 편안하게 사용할 수 있습니다.
- **마이크로 USB 케이블**. 마이크로 USB 케이블을 사용하여 충전을 위해 HoloLens 전원 공급 장치에 연결하거나 HoloLens 컴퓨터에 연결합니다.
- **전원 공급 장치**. 전원 출선에 연결합니다.
- **극세사 천**. 의류를 사용하여 HoloLens 바이저를 정리합니다.

>[!TIP]
>[클릭기는](hololens1-clicker.md) 별도의 상자에 HoloLens(1세대)와 함께 표시됩니다.

### <a name="power-supply-details"></a>전원 공급 장치 세부 정보

장치와 함께 제공되는 전원 공급 장치와 USB 케이블은 충전을 가장 잘 지원하는 메커니즘입니다. 전원 공급 장치는 18W 충전기입니다.  2A에서 9V를 제공합니다.

충전율과 속도는 장치가 실행되는 환경에 따라 다를 수 있습니다.

장치가 켜져 있는 동안 내부 배터리 충전율을 유지/향상하려면 최소한 15W 충전기에 연결해야 합니다.

## <a name="care-and-cleaning"></a>주의 하 고 정리

HoloLens를 신중 하 게 처리 합니다. HoloLens를 리프트 하 고, 운반 하 고, 조정 하려면 &mdash; 오버 헤드 스트랩을 없는 장치 arm을 사용 합니다. 센터를 먼지와 지문을 사용 하지 않고 유지 하 고 접촉 하지 않도록 합니다. 반복 되는 청소가 센터를 손상 시킬 수 있으므로 장치를 깨끗 하 게 유지 해 보세요.

HoloLens에서 클리너 또는 제품 통풍구를 사용 하지 말고 물에 직접 병합 하거나 물에 직접 적용 하지 마십시오.

센터를 청소 하려면 머리카락 렌즈 브러시 또는 전구 스타일 렌즈 blower에서 카멜식 나 goat 사용 하 여 먼지를 제거 합니다. 적은 양의 distilled 물로 마이크로 파이버 천을 moisten 하 고이를 사용 하 여 원형 동작에서 센터를 천천히 지울 수 있습니다.

헤드 밴드 및 장치 팔을 포함 하 여 나머지 장치를 청소 하려면 가벼운 soap 및 물이 있는 보풀이 없는 천으로 moistened를 사용 합니다. 다시 사용 하기 전에 HoloLens을 완전히 건조 하도록 합니다.

## <a name="finding-the-serial-number"></a>일련 번호 찾기

HoloLens (첫 번째 gen) 장치의 일련 번호는 오른쪽 렌즈 아래에 인쇄 됩니다.
1. 사용자가 원하는 대로 장치를 최대한 활용 하세요. 장치를 앞으로 이동 하 고 수직으로 연결 합니다.
1. 계속 해 서 렌즈를 사용 하 여 장치를 뒤집어 전환 하면 아래쪽이 표시 됩니다.
1. 이제 오른쪽 렌즈가 왼쪽에 표시 되 고, 대역의 렌즈 아래에는 일련 번호가 표시 됩니다. 

일련 번호는 연결된 PC를 통해 찾을 수도 있습니다.

1. 장치 연결
1. 파일 탐색기에서 **이 PC** 로 이동합니다.
1. 마우스 오른쪽 단추를 클릭하고 HoloLens 장치의 **속성** 을 선택합니다.
1. 그러면 장치의 일련 번호가 표시 됩니다. 아래 스크린샷에서와 같습니다.
