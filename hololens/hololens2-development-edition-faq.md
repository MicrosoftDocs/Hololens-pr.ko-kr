# HoloLens 개발 에디션 FAQ

## 일반 사항

### HoloLens 2 개발 판으로 얻을 수 있는 기능

HoloLens 2 개발 에디션에는 HoloLens 2, $500, Azure 크레딧, 3 개월 Unity Pro 평가판, 3 개월 Pixyz 플러그 인 평가판 등이 포함 됩니다.

### HoloLens 2와 HoloLens 2 개발 에디션 간의 차이점은 무엇 인가요? 

프로덕션 설정에서 HoloLens 2 (상업용 판)와 연결 하는 데 사용할 엔터프라이즈 응용 프로그램의 내부 빌드 및 테스트에만 HoloLens 2 개발 에디션을 사용할 수 있습니다. 개발 버전을 구입 하는 과정에서 Microsoft는 특정 할인 크레딧이 나 특정 Microsoft 또는 타사 서비스 또는 도구에 대 한 평가판 액세스를 제한 된 기간 동안 확인 하는 등의 특정 권리가 제공 됩니다. [자세히 알아보기](hololens2-options.md?tabs=developeredition#1-learn-about-the-development-edition)

### 모든 HoloLens 2 버전을 개발에 사용할 수 있나요?

예. 모든 버전의 HoloLens 2를 사용 하 여 앱을 개발할 수 있습니다.

### Azure 크레딧, 3 개월 Unity Pro 평가판, 3 개월 Pixyz 플러그인 평가판에서 $500을 어떻게 받을 수 있나요?

귀하의 할인 크레딧이 나 평가판은 제품 배송 기간 2 일 이내에 구매에 사용 되는 이메일에 전송 됩니다.

### 체험 크레딧을 얻으려면 "내 HoloLens 2 (장치 전용) 장치를 HoloLens 2 개발 판으로" 업그레이드 "할 수 있나요?

아니요, 장치 전송 계획을 제공 하지 않습니다. 모든 버전에서 다른 버전으로 전송할 수 있는 기능은 없습니다. 

### Unity Pro는 무엇 인가요? 

Unity Pro는 중소 규모의 스튜디오/기업에 대 한 실시간 3D 플랫폼입니다. 기술 및 비기술적 기술이 혼합 되어 있는 팀을 위해 설계 되었습니다. [자세히 알아보기](https://store.unity.com/products/unity-pro)

### Pixyz 플러그인 이란? 

Pixyz 플러그 인은 거의 모든 형식을 읽고 변환 하 고, 최적화 하 고, Unity에 배치 하 고, 실시간 응용 프로그램에 대 한 사용 준비가 완료 된 자산을 제공 하기 위해 단계적으로 진행 합니다. [자세히 알아보기](https://unity.com/products/pixyz)

### HoloLens 2 개발 버전용으로 융자를 사용할 수 있는 시기는 언제 인가요?

2021 초기에는 HoloLens 2 개발 에디션의 융자를 받을 수 있습니다.

### HoloLens 2 개발 에디션 권리에 대 한 질문과 연락 하는 사람은 누구 인가요?

권리가 관련 된 질문이 있는 경우 866-425-4709, 월요일 ~ 금요일, 6AM ~ 6PM, PST에 판매 전문가에 게 문의 하세요.

### 회사에서 여러 HoloLens 2 개발 에디션 장치를 구매할 수 있나요? 

이 장치는 mixed reality 솔루션을 빌드하는 개발자를 위한 것입니다. 고객 거래 당 구매 1 개를 제한 합니다. 회사에서 여러 개발 에디션 장치를 구입 하려는 경우 회사를 대신 하 여 HoloLens 2 개발 에디션 장치를 구입 하는 별도의 트랜잭션이 필요 합니다. 

### 내 HoloLens 2 개발 에디션 장치를 고객에 게 표시할 수 있나요? 

예,이 장치는 mixed reality 솔루션을 프로토타입 작성 및 테스트 하는 데 사용 됩니다.

### HoloLens 2 개발 에디션을 어디에서 구입할 수 있나요? 

HoloLens 2 개발 에디션은 Microsoft Store에 독점적으로 판매 됩니다.  

### 내 HoloLens 2 개발 에디션을 반환할 수 있나요?  

모든 판매가 최종 이며, HoloLens 2 개발 판에 사용할 수 있는 환불 없습니다.

### 개발자 에디션에는 HoloLens 2와는 다른 하드웨어 요구 사항이 달라 집니다.

아니요,이 장치는 하드웨어 요구 사항 측면에서 동일 합니다. 개발 에디션는 상업용 설정의 응용 프로그램 개발 및/또는 테스트 aiding에 배포 될 수 있습니다. 

### HoloLens 2 개발 판에서 개발 하는 데 Mixed Reality 툴킷 – Unity를 사용 하는 방법은 무엇 인가요?

MRTK-Unity은 Unity에서 플랫폼 간 MR을 개발 하는 데 사용 되는 구성 요소와 기능 집합을 제공 하는 Microsoft 구동 프로젝트입니다. [자세히 알아보기](https://microsoft.github.io/MixedRealityToolkit-Unity/README.html)

### HoloLens 2 개발 에디션에 개발 하는 데 Mixed Reality 도구 키트를 사용 하는 방법-리얼 UXT를 어떻게 사용할 수 있나요?

MRTK-Unreal는 첫 번째 UX 도구 구성 요소를 해제 하 고 현재 HoloLens 2 에서만 지원 됩니다. 구성 요소 플러그 인에는 입력 시뮬레이션에 대 한 일반적인 UX 기능에 대 한 코드, 청사진, 예제 자산, 직접 조작 배우, 눌러서 사용할 수 있는 단추 구성 요소, 조작자 구성 요소, 동작 구성 요소 팔 로우가 있습니다. [자세히 알아보기](https://microsoft.github.io/MixedReality-UXTools-Unreal/version/public/0.9.x/Docs/WelcomeToUXTools.html)

## Azure

### $500 Azure 크레딧을 어떻게 전환할 수 있나요?

$500 Azure 크레딧을 사용 하려면 Azure Pass 홍보용 코드를 사용 하 여 구독을 활성화 합니다. [자세히 알아보기](https://www.microsoftazurepass.com/Home/HowTo?Length=5)

Azure Pass [구독 약관](https://azure.microsoft.com/support/legal/subscription-agreement/?country=US) 을 검토 하 고 [세부 정보를 제공](https://azure.microsoft.com/en-us/offers/azure-pass/)합니다.

### 내 $500 Azure 크레딧이 만료 되는 경우는 언제 인가요?

Azure 키 코드를 활성화 한 후 $500 Azure 크레딧이 90 일 후에 만료 됩니다.

### Azure 크레딧을 사용 하 여 어떤 작업을 할 수 있나요?

Azure 크레딧이 있으면 Microsoft&#39;s//확장 클라우드 서비스 집합을 사용 하 여 구축할 수 있습니다. HoloLens 2 개발 판에 포함 된 Azure 제작진은 고객이 Azure에서 mixed reality 응용 프로그램을 빠르게 작성 하 고 테스트 하는 데 사용할 수 있도록 하는 것입니다. Azure에서 빌드하는 방법에 대 한 자세한 내용은 [azure 리소스](https://azure.microsoft.com/en-us/resources/) 를 참조 하세요.

### Azure mixed reality 서비스에 대 한 자세한 내용은 어디에서 확인할 수 있나요?

Microsoft는 개발자가 mixed reality 솔루션을 빌드하고 테스트 하는 데 사용할 수 있는 mixed reality 서비스 집합을 제공 합니다. [자세히 알아보기](https://docs.microsoft.com/windows/mixed-reality/develop/mixed-reality-cloud-services)
