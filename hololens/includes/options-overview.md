# [HoloLens 2 Development Edition](#tab/developeredition)

## 1. Development Edition에 대해 자세히

> [!NOTE]
> Development Edition은 상업용 배포 솔루션과 함께 배포할 수 없습니다. Development Edition은 상업적 설정에서 응용 프로그램의 개발 및/또는 테스트를 지원하기 위해 배포할 수 있습니다.  

HoloLens 2 Development Edition에는 HoloLens 2, Azure 크레딧 $500, 3개월 Unity Pro 평가판 및 3개월 PiXYZ 플러그 인 평가판이 포함되어 있습니다.

| 번들 기능 | 설명 |
|---|---|
|  [Unity](https://unity.com/) | 실시간 3D 개발 플랫폼입니다.   |
|  [Unity용 Pixyz 플러그 인](https://www.pixyz-software.com/plugin/) | 엔지니어링 CAD 월드와 개발 세계를 연결하는 &amp; 브리지입니다.   |
| [Azure 서비스에 대한 Azure 크레딧 $500](https://azure.microsoft.com/resources/) | HoloLens 2 Development Edition에 포함된 Azure 크레딧은 Azure에서 혼합 현실 응용 프로그램을 신속하게 구축하고 테스트하는 것입니다. |

모든 HoloLens 2 버전은 동일한 [](../hololens2-hardware.md#hololens-components) 하드웨어, 동일한 콘텐츠를 상자에 가지며 사전 설치된 동일한 [소프트웨어와 함께 제공합니다.](../hololens2-hardware.md#pre-installed-software)

## 2. 크레딧 사용

### Unity
고유한 코드를 받았을 것입니다. Unity Pro 평가판을 교환하는 길:
1. 웹 [id.unity.com](http://id.unity.com/)
1. Unity ID로 로그인하거나 하나 만들기
1. 제품 **코드 교환을 클릭합니다.**
1. 제품 코드를 입력하고 제품 **코드 교환을 클릭합니다.**

Unity 라이선스를 정품 [인증하는 방법에 대한 자세한 내용](https://support.unity3d.com/hc/articles/211438683-How-do-I-activate-my-license-) [필요한 경우 Unity 지원에](https://support.unity3d.com/hc) 문의하여 도움을 문의합니다.  

### Pixyz
고유한 코드를 받았을 것입니다. Pixyz 플러그 인 평가판을 교환하는 길:
1. 웹 [id.unity.com](http://id.unity.com/)
1. 제품 **코드 교환을 클릭합니다.**
1. Pixyz 플러그 인 제품 코드를 입력하고 제품 코드 **교환을 클릭합니다.**

[Pixyz](https://www.pixyz-software.com/documentations/html/2020.1/review/TrialLicense.html) 라이선스를 교환하는 방법에 대한 자세한 내용입니다. 필요한 [경우 Pixyz 지원에](https://www.pixyz-software.com/support/) 문의하여 도움을 문의합니다.

### Azure
다음은 Azure 크레딧을 교환하고자 할 수 있는 [혼합](https://azure.microsoft.com/topic/mixed-reality/) 현실에 대한 몇 가지 일반적인 Azure 제품입니다.
Azure 크레딧을 교환하는 방법: $500 Azure 크레딧을 교환하려면 Azure Pass 프로모션 코드를 교환하고 구독을 활성화합니다. [자세히 알아보세요.](../hololens2-development-edition-faq.md#how-can-i-redeem-my-500-azure-credit)

- [Azure 공간 앵커를 통해 3D 콘텐츠를 매핑, 공유 및 유지하는 앱을 빌드하는 방법에 대해 자세히 알아보겠습니다.](https://azure.microsoft.com/services/spatial-anchors/)
- [Azure 원격 렌더링이 고품질의 대화형 3D 콘텐츠를 렌더링하고 실시간으로 디바이스에 스트리밍하는 방법을 알아보겠습니다.](https://azure.microsoft.com/services/remote-rendering/)
