# [HoloLens 2 개발 판](#tab/developeredition)

## 1. 개발 판에 대 한 자세한 정보

> [!NOTE] 
> 개발 에디션는 상업적으로 분산 된 솔루션을 사용 하 여 배포할 수 없습니다. 개발 에디션는 상업용 설정의 응용 프로그램 개발 및/또는 테스트 aiding에 배포 될 수 있습니다.  

HoloLens 2 개발 에디션에는 HoloLens 2, $500, Azure 크레딧, 3 개월 Unity Pro 평가판, 3 개월 PiXYZ 플러그 인 평가판 등이 포함 됩니다.

| 번들 기능 | 설명 |
|---|---|
|  [Unity](https://unity.com/) | 실시간 3D 개발 플랫폼   |
|  [Unity 용 Pixyz 플러그 인](https://www.pixyz-software.com/plugin/) | 개발 환경에서 엔지니어링 CAD를 연결 하는 다리 &amp; 입니다.   |
| [Azure 서비스에 대 한 Azure 크레딧 $500](https://azure.microsoft.com/resources/) | HoloLens 2 개발 버전에 포함 된 Azure 제작진은 Azure에서 혼합 현실 응용 프로그램의 빌드 및 테스트를 빠르게 시작 하는 데 도움이 됩니다. |

모든 HoloLens 2 에디션에는 동일한 하드웨어를 [사용 하 고 있으며,](../hololens2-hardware.md#hololens-components) 동일한 [사전 설치 소프트웨어](../hololens2-hardware.md#pre-installed-software)와 함께 제공 됩니다.

## 2. 제작진 사용

### Unity
고유한 코드를 받았습니다. Unity Pro 평가판을 사용 하는 지침: 
1. [Id.unity.com](http://id.unity.com/) 방문 
1. Unity ID로 로그인 하거나 하나를 만드세요.
1. **제품 코드 사용을 클릭 합니다** . 
1. 제품 코드를 입력 하 고 **제품 코드** 사용을 클릭 합니다.

[Unity 라이선스를 활성화 하는 방법](https://support.unity3d.com/hc/articles/211438683-How-do-I-activate-my-license-)에 대해 자세히 설명 합니다. 필요에 따라 [Unity 지원](https://support.unity3d.com/hc) 에 도움을 요청 합니다.  

### Pixyz
고유한 코드를 받았습니다. Pixyz 플러그 인 평가판을 사용 하는 지침: 
1. [Id.unity.com](http://id.unity.com/) 방문
1. **제품 코드 사용을 클릭 합니다** .
1. Pixyz 플러그 인 제품 코드를 입력 하 고 **제품 코드** 사용을 클릭 합니다.

[Pixyz 라이선스를 사용](https://www.pixyz-software.com/documentations/html/2020.1/review/TrialLicense.html) 하는 방법에 대해 자세히 알아보세요. 필요한 경우 [Pixyz 지원](https://www.pixyz-software.com/support/) 에 문의 하 여 도움을 받으세요. 

### Azure
다음은 몇 가지 [혼합 현실 용 azure 제품](https://azure.microsoft.com/topic/mixed-reality/) 에 대 한 azure 크레딧을 사용 하는 것입니다.
Azure 크레딧을 사용 하는 방법: $500 Azure 크레딧을 사용 하려면 Azure Pass 홍보용 코드를 사용 하 여 구독을 활성화 합니다. [자세히 알아보세요.](../hololens2-development-edition-faq.md#how-can-i-redeem-my-500-azure-credit)

- [Azure 공간 앵커를 사용 하 여 3D 콘텐츠를 매핑하고, 공유 하 고, 유지 하는 앱을 빌드하는 방법을 알아봅니다.](https://azure.microsoft.com/services/spatial-anchors/)
- [Azure 원격 렌더링이 고품질의 대화형 3D 콘텐츠를 렌더링 하 고 실시간으로 장치에 스트리밍하는 방법에 대해 알아봅니다.](https://azure.microsoft.com/services/remote-rendering/)
