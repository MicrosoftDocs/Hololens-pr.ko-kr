# <a name="get-your-hololens-2-ready-to-use"></a>HoloLens 2 사용 준비

다음 절차는 처음으로 HoloLens 2를 설정하는 데 도움이 됩니다.

## <a name="charge-your-hololens"></a>HoloLens 충전

USB-C 케이블(포함됨)을 사용하여 전원 공급 장치를 충전 포트에 연결합니다. 전원 공급 장치를 전원 콘센트에 연결합니다. 장치와 함께 제공되는 전원 공급 장치와 USB-C-C 케이블이 HoloLens 2를 충전하는 가장 좋은 방법입니다. 충전기는 18W의 전력을 공급합니다(9V/2A). HoloLens 2 장치는 제공된 벽면 충전기를 사용하여 장치가 대기 중일 때 65분 이내에 배터리를 완전 충전할 수 있습니다.

충전율과 속도는 장치가 실행되는 환경에 따라 다를 수 있습니다.

- 장치가 충전 중일 때는 배터리 표시등이 켜져 있고 현재 충전 수준을 나타냅니다.  마지막 표시등은 페이드 인/페이드 아웃하여 활성 충전을 나타냅니다.
- HoloLens가 켜져 있으면 배터리 표시등에 배터리 잔량이 증분으로 표시됩니다.
- 5개의 표시등 중 하나만 켜져 있으면 배터리 잔량이 20% 미만입니다.
- 배터리 잔량이 매우 낮은 상태에서 장치를 켜려고 하면 표시등 1개가 잠깐 깜박인 다음 꺼집니다.

추가 정보가 필요한 경우 [장치 충전에 대한 자세한 내용은 여기에서 확인할 수 있습니다](hololens2-charging.md#charging-the-device). 

## <a name="adjust-fit"></a>핏 조정

HoloLens 2를 머리에 씁니다. 안경을 쓰고 있는 경우 착용한 상태로 둡니다.  브로우 패드가 이마에 편안히 놓여있고, 백 밴드는 머리의 뒤쪽 가운데 부분에 위치해야 합니다.

필요한 경우, 조정 휠을 돌려 헤드밴드를 확장한 다음 오버헤드 스트랩을 풉니다.

![HoloLens 2 핏 및 조정](images/hololens2-fit.png)

### <a name="attach-and-detach-the-overhead-strap"></a>오버헤드 스트랩 연결 및 분리

오버헤드 스트랩은 필수는 아니지만 장시간 사용 시 HoloLens 2 착용을 더욱 편안하게 해줄 수 있습니다.

오버헤드 스트랩의 앞쪽을 분리하려면 스트랩을 풀고 브로우 패드의 접이식 루프 사이로 밀어 넣습니다. 스트랩을 다시 연결하려면 루프를 당겨 다시 밀어 넣습니다.

오버헤드 스트랩의 뒤쪽을 분리하려면 각 연결 탭 아래 있는 단추를 누르고 가볍게 당깁니다. 다시 연결하려면 딸깍 소리가 날 때까지 연결 탭을 슬롯으로 다시 밀어 넣습니다.

![HoloLens 2 헤드 스트랩 연결 또는 제거](images/hololens2-headstrap.png)

## <a name="turn-on-the-hololens-2"></a>HoloLens 2 켜기

HoloLens 2를 켜려면 전원 단추를 누릅니다.  전원 단추 아래의 LED 표시등이 배터리 잔량을 표시합니다.

> [!NOTE]
> 처음으로 HoloLens 2의 전원을 켜려면 상자에서 꺼내서 전원 단추를 4초 이상 길게 눌러 켭니다. 다음에 HoloLens 2를 켤 때는 전원 단추를 짧게 누르면 시작됩니다.

### <a name="power-button-actions-for-different-power-transitions"></a>다른 전력 전환에 대한 전원 단추 동작

| 원하는 작업 | 이 작업 수행 | HoloLens 2의 동작 |
| - | - | - |
| 켜기 | 단추를 한 번 누릅니다. | 5개의 표시등이 모두 켜진 다음 변경되어 배터리 잔량을 표시합니다. 4초 후에 소리가 납니다. |
| 절전하기 | 단추를 한 번 누릅니다. | 5개의 표시등이 모두 켜졌다가 한 번에 하나씩 꺼집니다. 표시등이 꺼진 후 소리가 나고 화면에 "종료"가 표시됩니다. |
| 절전 모드 해제하기 | 단추를 한 번 누릅니다. | 5개의 표시등이 모두 켜진 다음 변경되어 배터리 잔량을 표시합니다. 즉시 소리가 납니다. |
| 끄기 | 5초 동안 길게 누릅니다. |  5개의 표시등이 모두 켜졌다가 한 번에 하나씩 꺼집니다. 표시등이 꺼진 후 소리가 나고 화면에 "종료"가 표시됩니다. |
| 응답이 없을 경우 HoloLens를 강제로 다시 시작하기 | 10초간 길게 누릅니다. | 5개의 표시등이 모두 켜졌다가 한 번에 하나씩 꺼집니다. 표시등이 꺼진 후입니다. |

## <a name="hololens-behavior-reference"></a>HoloLens 동작 참조

HoloLens의 표시등이 무슨 의미인지 확실하지 않나요? 충전하는 동안 HoloLens가 어떻게 동작해야 하는지 알고 싶으신가요?  다음은 도움이 되는 정보입니다!

### <a name="charging-behavior"></a>충전 동작

| 장치 상태 | 작업 | HoloLens 2의 동작 |
| - | - | - |
| OFF | USB 케이블 연결 | 장치가 켜짐으로 전환되고 배터리 표시등이 배터리 잔량을 나타내며 장치 충전이 시작됩니다.
| 켜기 | USB 케이블 제거 | 장치 충전 중지
| 켜기 | USB 케이블 연결 | 장치 충전 시작
| 절전 | USB 케이블 연결 | 장치 충전 시작
| 절전 | USB 케이블 제거 | 장치 충전 중지
| USB 케이블이 연결된 상태에서 켜짐 | 장치 끄기 | 장치가 켜짐으로 전환되고 배터리 표시등이 배터리 잔량을 나타내며 장치 충전 시작 |

### <a name="lights-that-indicate-the-battery-level"></a>배터리 잔량을 나타내는 표시등

| 표시등 수 | 배터리 잔량 |
| - | - |
| 4개의 표시등이 켜짐, 1개의 표시등이 페이드 인/아웃 | 100~81%(완전 충전됨) |
| 3개의 표시등 켜짐, 1개의 표시등 페이드 인/아웃 | 80~61% |
| 2개의 표시등 켜짐, 1개의 표시등 페이드 인/아웃 | 60~41% |
| 1개의 표시등 켜짐, 1개의 표시등 페이드 인/아웃 | 40~21% |
| 1개의 표시등 페이드 인/아웃 | 20~5% 이하(배터리 위험) |

### <a name="sleep-behavior"></a>절전 동작

| 장치 상태 | 작업 | HoloLens 2의 동작 |
| - | - | - |
| 켜기 | 전원 단추 한 번 누름 | 장치가 절전 모드로 전환하고 모든 표시등 꺼짐 |
| 켜기 | 3분 동안 동작 없음 | 장치가 절전 모드로 전환하고 모든 표시등 꺼짐 |
| 절전 | 전원 단추 한 번 누름 | 장치가 켜짐으로 전환하고 표시등 켜짐 |

### <a name="lights-to-indicate-problems"></a>문제를 나타내는 표시등

| 사용자가 이 작업을 수행하는 경우 | 표시등 동작 | 의미 |
| - | - | - |
| 전원 단추를 누릅니다. | 1개의 표시등이 5번 깜박이고 꺼집니다. | HoloLens 배터리가 매우 부족합니다. HoloLens를 충전합니다. |
| 전원 단추를 누릅니다. | 5개의 표시등이 5번 깜박이고 꺼집니다. |  HoloLens를 제대로 시작할 수 없고 오류 상태입니다. [운영 체제를 다시 설치](hololens-recovery.md)하여 장치를 복구합니다. |
| 전원 단추를 누릅니다. | 1번째, 3번째 및 5번째 표시등이 모두 연속적으로 깜박입니다. |  HoloLens에 하드웨어 오류가 있을 수 있습니다. [고객 지원](https://support.microsoft.com/en-us/supportforbusiness/productselection?sapid=3ec35c62-022f-466b-3a1e-dbbb7b9a55fb)에 문의하세요. |

## <a name="safety-and-comfort"></a>안전 및 편안함

### <a name="use-hololens-in-safe-surroundings"></a>안전한 환경에서 HoloLens 사용

장애물이나 넘어질 위험이 없는 안전한 공간에서 HoloLens를 사용합니다. 차량을 조작하거나 기타 잠재적으로 위험한 활동을 수행하는 등 명확한 시야가 필요하거나 전적으로 주의를 기울일 수 없는 경우에는 사용하지 마세요.

### <a name="stay-comfortable"></a>편안하게 유지

HoloLens를 사용하는 처음 몇 번의 세션은 짧게 하고 반드시 휴식을 취하세요. 불편함을 느낀다면, 멈추고 나아질 때까지 쉬세요. 일시적인 메스꺼움, 멀미, 현기증, 방향 감각 상실, 두통, 피로, 눈의 피로, 또는 건조한 눈 등이 이에 해당할 수 있습니다.

[제품 안전 경고 및 지침](https://support.microsoft.com/help/4558037/product-safety-warnings-and-instructions)을 참조하세요.
