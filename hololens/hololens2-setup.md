# HoloLens 2 사용 준비

아래 절차는 처음으로 HoloLens 2를 설정하는 데 도움이 됩니다.

## HoloLens 충전

USB-C 케이블(포함)을 사용하여 전원 공급 장치를 충전 포트에 연결합니다. 전원 공급 장치를 전원 콘센트에 연결합니다. 디바이스와 함께 제공되는 전원 공급 장치와 USB-C-to-C 케이블이 HoloLens 2를 충전하는 가장 좋은 방법입니다. 충전기는 18W의 전력을 공급합니다(2A에서 9V).

충전 속도는 디바이스가 실행되는 환경에 따라 다를 수 있습니다.

- 디바이스가 충전 중일 때 배터리 표시등이 켜지면서 현재 충전 수준을 나타냅니다.  마지막 표시등은 활성 충전을 나타내기 위해 안 페이드 인/페이드 아웃합니다.
- HoloLens가 켜져 있으면 배터리 표시등에 배터리 잔량이 점진적으로 표시됩니다.
- 5개의 표시등 중 하나만 켜져 있으면 배터리 잔량이 20% 미만입니다.
- 배터리 잔량이 매우 낮은 상태에서 장치를 켜려고 하면 한 개의 표시등이 잠깐 깜박인 다음 꺼집니다.

## 핏 조정

HoloLens 2를 머리에 놓습니다. 안경을 쓰신다면, 착용한 상태로 둡니다.  브로우 패드가 이마에 편안히 놓여있고, 뒷 밴드는 머리의 가운데 뒤에 위치해야 합니다.

필요한 경우 조정 휠을 돌려 헤드밴드를 확장한 다음 오버헤드 스트랩을 풉니다.

![HoloLens 2 핏 및 조정](images/hololens2-fit.png)

### 오버헤드 스트랩 연결 및 분리

오버헤드 스트랩은 필요하지 않지만, 장시간 사용 시 HoloLens 2를 더욱 편안하게 착용할 수 있습니다.

오버헤드 스트랩의 앞쪽을 분리하려면 스트랩을 풀고 브로우 패드의 접이식 루프 사이로 밀어 넣습니다. 이를 다시 연결하려면 루프를 당겨 스트랩을 다시 밀어 넣습니다.

오버헤드 스트랩의 뒤쪽을 분리하려면 각 연결 탭 아래의 단추를 누르고 부드럽게 당깁니다. 다시 연결하려면 클릭 소리가 날 때까지 연결 탭을 슬롯에 다시 밀어 넣습니다.

![HoloLens 2 헤드 스트랩 연결 또는 제거](images/hololens2-headstrap.png)

## HoloLens 2 켜기

HoloLens 2를 켜려면 전원 단추를 누릅니다.  전원 단추 아래의 LED 표시등에 배터리 잔량이 표시됩니다.

> [!NOTE]
> HoloLens 2의 전원을 처음으로 켜려면, 박스를 풀고 전원 버튼을 4초 이상 길게 눌러 켭니다. 다음에 HoloLens 2를 켤 때, 전원 단추를 짧게 누르면 시작됩니다.

### 다른 전원 전환에 대한 전원 단추 동작

| 수행할 작업 | 실행할 작업 | HoloLens 2에서 수행할 작업 |
| - | - | - |
| 켜려면 | 단추를 한 번 누릅니다. | 5개의 표시등이 모두 켜진 다음 배터리 잔량을 표시하도록 변경합니다. 4초 후에 소리가 납니다. |
| 절전하려면 | 단추를 한 번 누릅니다. | 다섯 개의 표시등이 모두 켜졌다가 한 번에 하나씩 꺼집니다. 표시등이 꺼진 후, 소리가 나고 화면에 "종료"가 표시됩니다. |
| 절전 모드를 해제하려면 | 단추를 한 번 누릅니다. | 5개의 표시등이 모두 켜진 다음 배터리 잔량을 표시하도록 변경합니다. 소리가 즉시 납니다. |
| 끄려면 | 5초 동안 길게 누릅니다. |  다섯 개의 표시등이 모두 켜졌다가 한 번에 하나씩 꺼집니다. 표시등이 꺼진 후, 소리가 나고 화면에 "종료"가 표시됩니다. |
| 응답이 없을 경우 HoloLens를 강제로 다시 시작하려면 | 10초간 길게 누르고 있습니다. | 다섯 개의 표시등이 모두 켜졌다가 한 번에 하나씩 꺼집니다. 표시등이 꺼진 후입니다. |

## HoloLens 동작 참조

HoloLens의 표시등이 무엇을 의미하는지 확실하지 않습니까? 충전하는 동안 HoloLens가 어떻게 작동하는지 알고 싶으신가요?  몇 가지 도움말을 참조하세요.

### 충전 동작

| 장치 상태 | 작업 | HoloLens 2가 이 작업을 수행합니다. |
| - | - | - |
| 끄기 | USB 케이블 연결 | 장치 전환이 켜지고 배터리 잔량을 나타내는 표시기 표시등이 표시되며 장치가 충전을 시작합니다.
| 켜기 | USB 케이블 제거 | 장치가 충전 중지
| 켜기 | USB 케이블 연결 | 장치가 충전 시작
| 절전 | USB 케이블 연결 | 장치가 충전 시작
| 절전 | USB 케이블 제거 | 장치가 충전 중지
| USB 케이블이 연결되어 켜짐 | 장치 끄기 | 장치 전환이 켜지고 배터리 잔량을 나타내는 표시기 표시등이 표시됨, 장치 충전이 시작됨 |

### 배터리 잔량을 나타내는 표시등

| 표시등 수 | 배터리 잔량 |
| - | - |
| 네 개의 켜진 표시등, 한 개의 표시등 페이드 인 및 아웃 | 100% ~ 81%(완전히 충전 됨) |
| 세 개의 켜진 표시등, 한 개의 표시등 페이드 인 및 아웃 | 80% ~ 61% |
| 두 개의 켜진 표시등, 한 개의 표시등 페이드 인 및 아웃 | 60% ~ 41% |
| 한 개의 켜진 표시등, 한 개의 표시등 페이드 인 및 아웃 | 40% ~ 21% |
| 한 개의 표시등 페이드 인 및 아웃 | 20% ~ 5% 사이 및 그 이하(배터리 위험) |

### 절전 동작

| 장치 상태 | 작업 | HoloLens 2가 이 작업을 수행합니다. |
| - | - | - |
| 켜기 | 단일 전원 단추 누름 | 절전 모드로 전환하여 모든 표시등을 끕니다. |
| 켜기 | 3분 동안 움직임 없음 | 절전 모드로 전환하고 모든 표시등을 끕니다. |
| 절전 | 단일 전원 단추 누름 | 장치 전환 및 표시등 켜기 |

### 문제를 나타내는 표시등

| 사용자가 이 작업을 수행하는 경우 | 표시등에서 수행하는 작업 | 의미 |
| - | - | - |
| 전원 단추를 누릅니다. | 한 표시등이 다섯 번 깜박이고 꺼집니다. | HoloLens 배터리가 매우 부족합니다. HoloLens를 충전합니다. |
| 전원 단추를 누릅니다. | 다섯 개의 표시등이 다섯 번 깜박이고 꺼집니다. |  HoloLens를 제대로 시작할 수 없으며 오류 상태입니다. 디바이스를 복구하려면 [운영 체제를 다시 설치](hololens-recovery.md)합니다. |
| 전원 단추를 누릅니다. | 1번째, 3번째 및 5번째 표시등이 연속적으로 깜박입니다. |  HoloLens에 하드웨어 오류가 있을 수 있습니다. 확인하려면 [OS를 다시 설치](hololens-recovery.md)하고 다시 시도하세요. OS를 다시 설치한 후에 표시등 깜박이 패턴이 계속되면 [지원 담당자](https://support.microsoft.com/en-us/supportforbusiness/productselection?sapid=3ec35c62-022f-466b-3a1e-dbbb7b9a55fb)에게 문의하세요. |

## 안전 및 편안함

### 안전한 환경에서 HoloLens 사용

장애물과 넘어질 위험이 없는 안전한 공간에서 HoloLens를 사용합니다. 차량을 조작하거나 기타 잠재적으로 위험한 작업을 수행하는 등 명확한 시야와 주의가 필요한 경우에는 사용하지 않습니다.

### 편안하게 유지

HoloLens와 처음 몇 번의 세션을 짧게 하고 반드시 휴식을 취하세요. 불편함을 느낀다면, 멈추고 나아질 때까지 휴식을 취합니다. 여기에는 일시적인 메스꺼움, 멀미, 현기증, 방향 감각 상실, 두통, 피로, 눈의 피로, 또는 건조한 눈 등이 해당합니다.

[제품 안전 경고 및 지침](https://support.microsoft.com/help/4558037/product-safety-warnings-and-instructions)을 참조하세요.
